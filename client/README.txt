Необходимая версия питона - python3

(мне не удалось скомпилировать клиент в исполняемый файл, поэтому для запуска придется запускать свою среду.)

Необходимо установить среду:
py -m venv .env               На случай, если папки .env нет
.env\Scripts\activate         Запуск виртуального окружения

Установить зависимости:
pip install -r requirements.txt



Для запуска
.\.env\Scripts\activate       Если перед терминалом не появилась надпись (.env)
py main.py                    Запускает сам клиент
